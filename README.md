# VPN Bot — Telegram-бот для продажи VPN-подписок через Stars

[![Go Version](https://img.shields.io/badge/Go-1.25-blue)](https://go.dev/)
[![Telegram Bot API](https://img.shields.io/badge/Telegram%20Bot%20API-v5.5.1-blue)](https://github.com/go-telegram-bot-api/telegram-bot-api)

Telegram-бот для автоматической продажи VPN-подписок с оплатой через **Telegram Stars (XTR)**. Бот интегрируется с **Marzban** — панелью управления прокси-серверами на базе Xray/VLESS — и полностью автоматизирует цикл: от регистрации пользователя до истечения подписки.

## Возможности

- **Пробный период** — 4 дня бесплатного доступа для новых пользователей
- **Оплата Telegram Stars** — нативные платежи Telegram без сторонних процессингов
- **Несколько тарифов** — 1 месяц, 3 месяца, 6 месяцев, 1 год
- **Автоматическое создание VPN-пользователей** — через Marzban API (VLESS + Xray)
- **Реферальная программа** — 5-уровневая система вознаграждений (до 48% от покупки)
- **Уведомления об истечении** — напоминания за 3 дня, за 1 день и после истечения
- **Отслеживание трафика** — просмотр оставшегося трафика и дней подписки
- **Docker-ready** — полная контейнеризация с PostgreSQL

## Как это работает

```
Пользователь → Telegram Bot → Telegram Stars Invoice
                                    ↓
                            Оплата Stars
                                    ↓
                        Бот получает подтверждение
                                    ↓
              Marzban API → Создание VLESS-пользователя
                                    ↓
                    Пользователь получает ссылку-подписку
                                    ↓
                        Фоновая проверка раз в 6 часов
                                    ↓
              Уведомления о скором истечении / блокировка
```

1. Пользователь запускает бота командой `/start`
2. Бот создаёт запись в БД, проверяет реферальный код (если есть)
3. Пользователь выбирает: пробный период, покупка подписки или просмотр аккаунта
4. При покупке бот создаёт инвойс в Telegram Stars
5. После оплаты бот создаёт пользователя в Marzban с VLESS-конфигурацией
6. Раз в 6 часов бот проверяет истекающие подписки и отправляет уведомления
7. По истечении подписки пользователь теряет доступ к VPN

## Установка и запуск

### Требования

- Linux-сервер (рекомендуется Ubuntu 22.04+)
- Docker и Docker Compose v2
- Telegram Bot Token (получить у [@BotFather](https://t.me/BotFather))
- Установленный и настроенный [Marzban](https://github.com/Gozargah/Marzban)
- Кошелёк Telegram Stars для бота (настраивается у BotFather)

### Быстрый старт (Docker Compose)

1. **Клонируйте репозиторий:**

```bash
git clone https://github.com/your-username/vpn-bot.git
cd vpn-bot
```

2. **Создайте файл `.env`:**

```env
BOT_TOKEN=1234567890:ABCdefGHIjklmNOPqrstUVwxyz
MARZBAN_URL=https://your-marzban.com
MARZBAN_USERNAME=admin
MARZBAN_PASSWORD=your_marzban_password
DATABASE_URL=postgres://postgres:your_db_password@postgres:5432/vpn_bot?sslmode=disable
POSTGRES_PASSWORD=your_db_password
POSTGRES_DB=vpn_bot
POSTGRES_USER=postgres
```

3. **Запустите:**

```bash
docker compose up -d
```

4. **Проверьте логи:**

```bash
docker compose logs -f bot
```

### Ручной запуск

1. Установите PostgreSQL и создайте базу данных
2. Примените схему: `psql -d vpn_bot -f init.sql`
3. Установите переменные окружения (или используйте `.env`)
4. Запустите:

```bash
go mod download
go run .
```

## Схема базы данных

### `users`
Пользователи бота. Ключевые поля: `telegram_id`, `marzban_uuid`, `is_active`, `expire_at`, `data_limit`, `data_used`, `referrer_id`, `balance`, `trial_used`.

### `subscriptions`
Подписки пользователей. Содержит тарифный план, сумму, статус, даты начала/окончания.

### `payments`
Платежи в Telegram Stars. Трекинг от создания инвойса до подтверждения оплаты.

### `referrals`
Реферальные связи. Фиксирует, кто кого пригласил, уровень в Pyramid и комиссию.

### `notifications`
История отправленных уведомлений (чтобы не дублировать).

## Реферальная система

5-уровневая система вознаграждений с прогрессивным понижением процента:

| Уровень | Процент от покупки |
|---------|-------------------|
| 1-й уровень (напрямую) | 30% |
| 2-й уровень | 10% |
| 3-й уровень | 5% |
| 4-й уровень | 2% |
| 5-й уровень | 1% |
| **Максимальная выплата** | **48%** |

- Пригласительная ссылка: `https://t.me/your_bot?start=ref_<telegram_id>`
- Полученные Stars зачисляются на баланс и используются для покупки подписки
- После успешной покупки рефера пригласивший получает уведомление о начислении

## API Marzban

Бот использует [Marzban REST API](https://github.com/Gozargah/Marzban) для управления VPN-пользователями:

- **Аутентификация** — получение Bearer-токена (действителен 23 часа, автообновление)
- **`POST /api/user`** — создание пользователя с UUID, сроком действия и лимитом данных
- **`GET /api/user/{username}`** — получение информации о пользователе (трафик, статус)
- **Потокобезопасность** — `sync.Mutex` для конкурентного доступа к токену при обработке множества запросов

Создаваемые пользователи используют протокол **VLESS**, который автоматически настраивается через Marzban.
